Mahalo Knights

This project is a sample store layout for the Mahalo Knights fine clothing company. It is built using the MEAN stack (Mongo, Express, Angular, Node) and is a work in progress, with a plan to fully launch as a store sometime in 2016.

Skavengr

This project lets you organize and invite people on scavenger hunts anywhere in the world (called "quests"). It uses google maps to place markers that the quest creator can then write a clue for and specify a time window for the quest to take place. Participants use the app on their phones to track when they discover a location and receive a notification upon completion. The first person to finish all the clues gets a virtual trophy! Built with Ruby and Rails

Dinosaur Adventure

A simple educational game built using React with Haylee Nighbert. Hover your mouse over a dinosaur to see its hunger level and click to feed it. If you let any of the hunger levels reach zero, you'll be eaten!!! After being consumed, you can learn more about facts about the dinosaurs. Refresh the page to restart.
